Toast

Injectable service for toast notifications



Description

ToastService is an injectable service that displays temporary notification messages.
Toasts appear from the bottom-right corner with smooth animations and auto-dismiss after a specified duration.
Multiple toasts stack vertically with 16px spacing between them.
Toasts have max-width: 260px, min-height: 150px and expand based on content.
Click any toast to dismiss it early.

HTML

TypeScript


Methods:

  • toast(options: ToastOptions) - Displays a success/info toast with default styling (border: var(--nc-toast-border), background: var(--nc-toast-bg))
  • error(options: ToastOptions) - Displays an error toast with error styling (border: var(--nc-error-toast-border), background: var(--nc-error-toast-bg))

ToastOptions Interface:

  • text (string, required) - The message to display in the toast
  • duration (number, optional, default: 2500) - Duration in milliseconds before the toast auto-dismisses

Features:

  • ✓ Fixed position at bottom-right corner (right: 20px, bottom: 20px)
  • ✓ Smooth slide-in/slide-out animations (300ms transition)
  • ✓ Multiple toasts stack vertically with automatic repositioning
  • ✓ Click to dismiss any toast early
  • ✓ Dynamic DOM creation - no template elements needed
  • ✓ Max width: 260px, Min height: 150px (expands with content)
  • ✓ Injectable service - use anywhere in your application


Collapsible

Expandable content component



Description

Collapsible component allows you to show/hide content with smooth animations.
It requires a trigger element (slot="collapsible-trigger") that toggles the visibility when clicked.
The component is 100% wide and expands/collapses with a 0.3s transition.
Can be controlled programmatically via public methods or through callbacks.
