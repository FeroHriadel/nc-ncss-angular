Exposed methods:

  • getValue() - Returns the current input value (string)
  • setValue(value: string) - Sets the input value
  • clear() - Clears the input value


Attributes list:

  • class (string, default: '') - CSS classes to apply to the wrapper
  • style (object, default: {}) - Inline styles object for the wrapper
  • inputClass (string, default: '') - CSS classes to apply to the input element
  • inputStyle (object, default: {}) - Inline styles object for the input element
  • id (string, default: '') - HTML id attribute
  • width (string, default: '') - Width of the component
  • onClick (function, default: () => {}) - Click handler for the wrapper
  • placeholder (string, default: '') - Placeholder text for the input
  • disabled (boolean, default: false) - Disables the input
  • value (string, default: '') - Initial value of the input
  • onChange (function, default: () => {}) - Callback function triggered when input value changes
  • name (string, default: '') - HTML name attribute
  • minLength (number, optional) - Minimum length validation
  • maxLength (number, optional) - Maximum length validation
  • required (boolean, default: false) - Makes the input required
  • defaultValue (string, default: '') - Default value for the input
