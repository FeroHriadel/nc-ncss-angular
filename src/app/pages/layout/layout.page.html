Card

Card component for displaying content in a structured format



Description

Card is a versatile container component that provides a clean, bordered box for displaying grouped content.
It includes built-in styling with background color, border, border-radius, and padding, making it perfect for organizing and presenting information in a structured way.

Card Title

This is a card component that provides a clean, bordered container for your content. Cards are perfect for grouping related information such as user profiles, product details, or summary sections.

You can customize the card's width, add custom styles, and apply CSS classes for padding, margins, and other styling needs.

HTML

TypeScript

Attributes list:

  • width (string, default: '100%') - Custom width (e.g., '500px', '50%', 'auto')
  • style (object, default: {{ '{' }}{{ '}' }}) - Custom CSS styles as key-value pairs
  • class (string, default: '') - CSS class names to apply (e.g., 'p-4' for padding)
  • id (string, default: '') - HTML id attribute for the element
  • onClick (function, default: () => {{ '{' }}{{ '}' }}) - Click event handler for making cards interactive
