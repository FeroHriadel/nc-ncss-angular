Exposed methods:

  • setValue(value: string | string[]) - Set the value of the select component
  • getValue() - Gets the selected value (string or string[] for multiselect)
  • clear() - Clears any selected value(s)
  • getOpen() - Returns whether the dropdown is currently open
  • open() - Open the dropdown programmatically
  • close() - Close the dropdown programmatically
  • toggle() - Toggle the dropdown open/closed programmatically
