Exposed methods:

  • setValue(value: string | string[]) - Set the value of the select component
  • getValue() - Gets the selected value (string or string[] for multiselect)
  • clear() - Clears any selected value(s)
  • getOpen() - Returns whether the dropdown is currently open
  • open() - Open the dropdown programmatically
  • close() - Close the dropdown programmatically
  • toggle() - Toggle the dropdown open/closed programmatically


Attributes list:

  • customTrigger (boolean, default: false) - When true, allows you to provide custom HTML content as the trigger instead of the default button
  • placeholder (string, default: '') - Text shown when no option is selected
  • options (SelectOption[], default: []) - Array of options with { value: string, label: string } format
  • width (string, default: '') - Sets the width of the select component (e.g., '200px', '100%')
  • optionsWidth (string, default: '') - Sets the width of the dropdown options container
  • multiple (boolean, default: false) - Enables multi-select mode allowing multiple options to be selected
  • disabled (boolean, default: false) - Disables the select component
  • defaultValue (string | string[], default: '') - Initial value(s) - single string or array of strings for multi-select
  • onChange (function, default: () => {}) - Callback function triggered when selection changes, receives the new value(s) as parameter
  • class (string, default: '') - CSS classes to apply to the select wrapper
  • style (object, default: {}) - Inline styles object to apply to the select wrapper
  • inputClass (string, default: '') - CSS classes to apply to the input element
  • inputStyle (object, default: {}) - Inline styles object to apply to the input element
  • optionsStyle (object, default: {}) - Inline styles object to apply to the options dropdown container
  • name (string, default: '') - HTML name attribute for the select
  • id (string, default: '') - HTML id attribute for the select
  • ariaLabel (string, default: name or 'Select an option') - ARIA label for accessibility
  • searchable (boolean, default: false) - When true, adds a search input inside the dropdown to filter options
  • label (string, default: '') - Optional label text displayed above the select component



Native Select

You may use the native select element for simple dropdowns. It has been styled for consistency with the nc-select component.
