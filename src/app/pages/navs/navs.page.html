Top Navigation

Common use and modifications



Description

TopNav is intended to be on the top of the page, providing easy navigation for users.
It shows logo (if provided) on the left. There's links on the right side, which can be simple links or dropdowns with options (nested links).

You can also add custom content to the right side using the "custom" slot, which is perfect for things like theme switchers, user profile menus, or any other custom elements you want to include in the navigation bar.

The component is responsive but is not inteded to show a billion links. It is developer's responsibility to use a reasonable amount of links on the right. From 768px and below a hamburger menu replaces the links.
