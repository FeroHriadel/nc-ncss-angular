Exposed methods:

  • getFiles() - Returns the selected file(s): File, File[], or null
  • clearFileInput(event: Event) - Clears the selected file(s)
  • clickFileInput() - Programmatically opens the file picker dialog


Attributes list:

  • class (string, default: '') - CSS classes to apply to the wrapper
  • style (object, default: {}) - Inline styles object for the wrapper
  • inputClass (string, default: '') - CSS classes to apply to the input element
  • inputStyle (object, default: {}) - Inline styles object for the input element
  • id (string, default: '') - HTML id attribute
  • width (string, default: '') - Width of the component
  • placeholder (string, default: '') - Placeholder text displayed when no file is selected
  • disabled (boolean, default: false) - Disables the file input
  • onChange (function, default: () => {}) - Callback function triggered when file selection changes
  • name (string, default: '') - HTML name attribute
  • accept (string, default: '*') - File types to accept (e.g., 'image/*', '.pdf', etc.)
  • multiple (boolean, default: false) - Allow multiple file selection
