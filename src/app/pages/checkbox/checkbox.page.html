HTML

TypeScript

Exposed methods:

  • getValue() - Returns the current checked state (boolean)
  • setValue(value: boolean) - Sets the checked state
  • toggleChecked() - Toggles the checked state


Attributes list:

  • class (string, default: '') - CSS classes to apply to the checkbox wrapper
  • style (object, default: {}) - Inline styles object for the wrapper
  • inputClass (string, default: '') - CSS classes to apply to the input element
  • inputStyle (object, default: {}) - Inline styles object for the input element
  • id (string, default: '') - HTML id attribute
  • onChange (function, default: undefined) - Callback function triggered when checkbox state changes
  • disabled (boolean, default: false) - Disables the checkbox
  • name (string, default: '') - HTML name attribute
  • checked (boolean, default: false) - Initial checked state
  • label (string, default: '') - Optional label text displayed next to checkbox
