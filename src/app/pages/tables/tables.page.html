Basic Virtualized Table

A performant table with built-in controls, filtering, and sorting

Description

The Virtualized Table component provides a complete solution for displaying large datasets with filtering, sorting, column visibility controls, and zoom functionality. It uses virtual scrolling for optimal performance with large data sets.

Result



Features

  • Filtering: Advanced filtering with multiple conditions (equals, contains, greater than, etc.)
  • Sorting: Click column headers to sort ascending/descending
  • Column Visibility: Show/hide columns dynamically
  • Zoom: Adjust table content size from 50% to 150%
  • Virtual Scrolling: Efficient rendering of large datasets
  • Filter Presets: Save and apply common filter combinations
  • Drag & Drop: Reorder columns by dragging headers
  • Striped Rows: Alternating row colors for better readability
  • Hover Effects: Highlight rows on hover
